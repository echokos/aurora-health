# Aurora Health Monitor Dashboard

Live health monitoring dashboard for Aurora's infrastructure.

**URL:** https://health.kos.plus (once deployed)

## Architecture

- **Static site** deployed on Cloudflare Pages
- **Auto-updates** via git push (health.json updated by cron)
- **Frontend:** DaisyUI + Alpine.js + Tailwind CSS
- **No build step** required - pure HTML/CSS/JS

## Files

- `dist/index.html` - Dashboard UI
- `dist/health.json` - Health check results (auto-updated every 5 minutes)

## Health Check System

Health data generated by `~/aurora/scripts/health-check.py` running every 5 minutes via cron.

**Components monitored:**
- Cron Jobs (6)
- Services (6)
- API Integrations (5)
- Memory System (4)
- Dashboards (2)

Total: 23 components across 5 groups

## Development

```bash
# Clone
git clone https://github.com/echokos/aurora-health.git
cd aurora-health

# Test locally
cd dist
python3 -m http.server 8000
# Visit http://localhost:8000
```

## Deployment

Automatic via Cloudflare Pages:
1. Push to main branch
2. Cloudflare auto-deploys within ~60 seconds
3. Changes live at health.kos.plus
